# Info

### 사용법
- 플레이어를 추가하고 각각 힘을 설정합니다.

- 해당 플레이어들이 다 함께 들 수 있는 몬스터들을 표시합니다.

- 플레이어 아이콘을 클릭하면 생존 여부를 변경할 수 있습니다.

- 사망한 플레이어는 계산 시 제외합니다.

.
### 무게 계산 공식  
- `5N + P >= W`  
  N : 플레이어 수  
  P : 힘 총합  
  W : 몬스터 무게
